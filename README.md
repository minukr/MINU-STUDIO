# MINU-STUDIO
파이썬과 파이게임으로 만든 간단한 작곡 프로그램
아직 많이 미완성입니다

## 사용법
1. 릴리즈에서 exe 파일을 다운받거나 main.py 실행 (pygame 모듈 있어야함)
2. 빈 json 파일 열기
3. 노래를 만드세요

중요! exe 파일, src 폴더, sounds 폴더가 같은 경로에 있어야 합니다

방화벽에서 뭐라 그러는건 무시하세요
바이러스 아닙니다

## 단축키
- 1, 2, 3 으로 악기를 바꿀 수 있습니다
-  MET 버튼을 눌러 메트로놈을 킬 수 있습니다
- CTRL을 누른 채로 드래그하여 노트를 여러개 선택할 수 있습니다
- CTRL S로 저장할 수 있습니다
